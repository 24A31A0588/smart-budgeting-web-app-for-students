💰 BudgetWise — Student Finance Tracker

A modern and responsive web application that helps students manage their finances effectively. Track expenses, monitor income, visualize data, and achieve savings goals — all in one place.

🚀 Overview

BudgetWise is a complete personal finance dashboard built for students. It allows users to manage daily transactions and gain insights into their spending habits using interactive charts.

📌 Features

🔐 Authentication UI

- Login and Signup pages
- Demo account option
- Local storage-based login

📊 Dashboard

- Total balance, income, and expenses
- Savings rate calculation
- Pie chart (category-wise expenses)
- Bar chart (monthly overview)
- Recent transactions

💸 Expense Management

- Add, edit, delete expenses
- Category filtering
- Search functionality
- Monthly filtering

💰 Income Tracking

- Add and manage income
- Multiple income sources

🎯 Savings Goals

- Create financial goals
- Track progress with progress bar
- Set target date

📈 Monthly Summary

- Month-wise financial overview
- Category-wise expense breakdown

✨ UI Features

- Dark theme UI
- Smooth animations
- Responsive design (mobile + desktop)

🛠️ Tech Stack
- HTML
- CSS
- JavaScript
